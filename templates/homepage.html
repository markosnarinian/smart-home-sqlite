About Narinian Smart Home

With Narinian Smart Home you can create DIY smart devices such as light bulbs, locks... Also, you can use Narinian Smart Home to use these devices across different services (Google Home, SmartThings)!

Steps to create a smart device

  • Log in and navigate to Devices
  • Add a new device from the Devices page
  • Download and install the host application
  • Link the host computer to your account
  • Configure the host appliction (e.g. Select actions)
